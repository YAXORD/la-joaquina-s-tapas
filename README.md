# 🥘 La Joaquina's Tapas - Management App

An interactive web application designed to modernize the hospitality experience, featuring a digital menu and efficient order management tailored for a traditional Spanish tapas bar.

This project was initially structured using **Lovable** (AI) to accelerate UI development, providing a rapid prototype to focus on refining the user experience (UX) and establishing a scalable technical foundation.

## 🚀 Key Features

*   **Digital Menu:** Real-time display of tapas, drinks, and daily specials.
*   **Order Management:** Streamlined system for taking orders and routing them efficiently to the kitchen.
*   **Optimized Interface:** Fluid, responsive design built specifically for tablets and mobile devices during active service.

## 🛠️ Tech Stack

*   **Frontend:** React, TypeScript, Tailwind CSS.
*   **Backend Base:** Supabase.
*   **Tooling:** Vite, Lovable.

## ⚙️ Local Installation & Setup

To run this project locally, execute the following commands in your terminal:

```bash
git clone [https://github.com/YAXORD/la-joaquina-s-tapas.git](https://github.com/YAXORD/la-joaquina-s-tapas.git)
cd la-joaquina-s-tapas
npm install
npm run dev
````
